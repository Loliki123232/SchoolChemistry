## 🎯 О проекте

**"Химия: школьный помощник"** — это десктопное приложение для изучения химии, разработанное в рамках учебного проекта. Приложение предоставляет доступ к учебным темам, интерактивной периодической таблице Менделеева и поиску информации о химических элементах.

---

## ⚙️ Функциональность

### 1. Просмотр учебных тем

- Автоматическая загрузка списка тем из базы данных при запуске
- Отображение первой темы по умолчанию
- Быстрая навигация между темами через боковое меню
- Форматированное отображение текста с поддержкой химических формул
<img width="879" height="600" alt="image" src="https://github.com/user-attachments/assets/9cfeedd3-a458-4286-81a2-56322d8ef0b7" />

### 2. Периодическая таблица Менделеева

- Отображение всех 118 химических элементов
- Группировка по периодам (1-7)
- Цветовая кодировка по типам элементов:
  - 🔴 Щелочные металлы
  - 🟡 Щёлочноземельные металлы
  - 🟢 Переходные металлы
  - 🔵 Галогены
  - ⚪ Инертные газы
- Всплывающие подсказки с детальной информацией об элементе
- Отдельные секции для лантаноидов и актиноидов

### 3. Поиск химического элемента

- Поиск по химическому символу (H, O, Fe, Au)
- Поиск по русскому названию (Водород, Кислород, Железо)
- Отображение детальной карточки элемента:
  - Атомный номер
  - Атомная масса
  - Группа и период
  - Описание и интересные факты
- Обработка некорректного ввода

## 💻 Технологии

| **WPF** | — | Фреймворк для UI |
| **PostgreSQL** | 14+ | База данных |
